Description du projet :


Examen 2026 de DevOps 

Cet examen contient donc une function de validation d'adresse email 

La validation test ces différents points =>

Un mail doit contenir au moins un @
Un mail doit contenir au moins un point dans le nom de domaine
Un mail ne peut pas contenir d'espace
Un mail ne peut pas etre vide avant ou aprés @

Et pour lancer les test il suffit d'éxecuter la commande "" npm test "" dans le terminal de VSCode
